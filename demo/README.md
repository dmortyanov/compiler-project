# MiniCompiler Showcase

Эта программа (`showcase.src`) демонстрирует все возможности компилятора, реализованные в рамках Спринтов 1-8.

## Как запустить

Вы можете использовать подготовленный bash-скрипт для автоматической сборки, линковки и запуска демо-программы:

```bash
chmod +x run_demo.sh
./run_demo.sh
```

## Что демонстрируется?

- **Sprint 1-3:** Базовые типы, переменные, выражения, логические операторы.
- **Sprint 4-6:** Управляющие конструкции (`if`, `for`, `while`), рекурсия (факториал, Фибоначчи).
- **Sprint 7:** Массивы (инициализация, доступ, передача в функции), сортировка QuickSort, внешние функции (вызов `printf` из libc).
- **Sprint 8:** Инлайнинг функций, оптимизации (propagation, fold, dead code).
