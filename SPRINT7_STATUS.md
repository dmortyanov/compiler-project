# Статус Спринта 7: Массивы, внешние вызовы, оптимизации

## Выполненные задачи:
1. **Поддержка массивов (ARR)**:
   - Синтаксис: статические массивы, инициализаторы `{1, 2, 3}`, доступ по индексу.
   - Выделение памяти на стеке `ALLOCA` и доступ `LOAD_ELEM`/`STORE_ELEM`.
2. **Внешние вызовы (EXT)**:
   - Ключевое слово `extern` и поддержка вызовов C-библиотек (например, `printf`).
   - Соответствие System V AMD64 ABI (передача аргументов в 64-битные регистры, вариативные функции).
3. **Оптимизации (OPT)**:
   - Конвейер оптимизатора: свёртка констант, CSE (Common Subexpression Elimination), Copy Propagation, DCE (Dead Code Elimination).
   - Интеграция в CLI (`--optimize`).
